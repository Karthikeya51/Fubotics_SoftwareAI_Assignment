Frontend (Vite + React)

Setup (local):

1. Ensure Node.js (>=16) and npm installed.
2. cd frontend
3. npm install
4. DEV: VITE_API_URL environment variable can be set to backend URL (default http://localhost:8000)
   - On Linux/macOS: export VITE_API_URL="http://localhost:8000"
   - On Windows (powershell): $env:VITE_API_URL="http://localhost:8000"
5. Run: npm run dev
